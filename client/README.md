# 🎨 Threads Clone - Frontend

Radhe Radhe, I am Pranita
This is the frontend for a Threads Clone application built using React (Vite) and Material UI. It connects to the backend API to provide a modern social media user experience.

---

## 🛠️ Tech Stack

* React (Vite)
* Material UI
* React Redux
* React Router

---

## 📦 Installation

Clone the repository and install dependencies:

```id="6cavv6"
git clone <your-repo-url>
cd frontend
npm install
```

---

## ▶️ Running the App

```id="q6ck7i"
npm run dev
```

App will run on:

```id="9q51fa"
http://localhost:5173
```

---

## 🔗 Backend Connection

Currently, the frontend directly connects to the backend using a hardcoded API URL inside the code.

Example:

```id="7n4y7w"
http://localhost:5000/api
```

---

## ✨ Features

* User Authentication UI
* Create & View Posts
* Responsive UI with Material UI
* Routing with React Router
* API integration with backend

---

## ⚠️ Note

For better scalability and security, it is recommended to use environment variables instead of hardcoding API URLs.

Example (optional improvement):

```id="3y7y92"
VITE_API_URL=http://localhost:5000
```
